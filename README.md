# 🌈 Rainbow Mode (Voidloop Mod)

Adds a **Rainbow Mode** toggle to the game's settings that overlays the screen with smoothly cycling rainbow colors.

Made using **BepInEx**.

---

## ✨ Features

- Adds **Rainbow Mode** option to Settings  
- Smooth animated rainbow screen effect  
- Saves automatically between game launches  

---

## 📦 Requirements

You must have:

- **Voidloop Demo**
- **BepInEx 5 (x64)**

Download BepInEx here:

👉 https://github.com/BepInEx/BepInEx/releases  

Download:
BepInEx_x64_5.x.x.zip

---

## 🔧 Installation

### 1. Install BepInEx

Open your game folder.

**Steam users:**
Steam Library
→ Right click Voidloop Demo
→ Manage
→ Browse local files

Extract the BepInEx zip into this folder.

You should now see:
Voidloop Demo/
BepInEx/
winhttp.dll
doorstop_config.ini


Launch the game **once**, then close it.

This creates the required folders.

---

### 2. Install Rainbow Mode

Download **RainbowMode.dll** from the Releases page.

Place it into:
Voidloop Demo/BepInEx/plugins/


Final structure:
Voidloop Demo/
BepInEx/
plugins/
RainbowMode.dll


---

### 3. Launch the Game

Start the game normally.

Open:
Settings → General:


Enable:
Rainbow Mode


🌈 Enjoy.

---

## ❓ Troubleshooting

### Mod not appearing

- Make sure the **BepInEx console** opens when launching the game.
- Check the DLL is inside the **plugins** folder, not the main folder.

### Nothing happens when enabled

- Try restarting the game after installing.

---

## ⚠️ Notes

- This mod only changes visuals.
- No gameplay changes.
- Safe to remove at any time.

To uninstall, simply delete:
BepInEx/plugins/RainbowMode.dll
