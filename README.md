# 🌈 Rainbow Mode (Voidloop Mod)

Adds a **Rainbow Mode** toggle to the game's settings that overlays the screen with smoothly cycling rainbow colors.

Made using **BepInEx**.

---

## ✨ Features

- Adds **Rainbow Mode** option to Settings  
- Smooth animated rainbow screen effect  
- Saves automatically between game launches  

---

## 📦 Requirements

- **Voidloop Demo**

No extra downloads required — **BepInEx is already included** in this release.

---

## 🔧 Installation

### 1. Download the Mod

Download the latest release from the **Releases** page.

Extract the downloaded `.zip`.

---

### 2. Install

Open your game folder.

**Steam users:**

Steam Library  
→ Right click **Voidloop Demo**  
→ Manage  
→ Browse local files  

---

Drag **all files from the download** into the game folder.

Allow Windows to merge folders if prompted.

Final structure should look like:


Voidloop Demo/
BepInEx/
winhttp.dll
doorstop_config.ini


---

### 3. Launch the Game

Start the game normally.

The BepInEx console should open automatically.

Open:


Settings → General


Enable:


Rainbow Mode


🌈 Enjoy.

---

## ❓ Troubleshooting

### Mod not appearing

- Make sure the **BepInEx console window** appears when launching the game.
- Ensure you copied the files directly into the **game folder**, not inside another folder.

### Nothing happens when enabled

- Restart the game after installing.
- Make sure old versions of the mod are removed.

---

## ⚠️ Notes

- This mod only changes visuals.
- No gameplay changes.
- Safe to remove at any time.

To uninstall, delete:


BepInEx/
winhttp.dll
doorstop_config.ini


(or simply verify game files through Steam).
